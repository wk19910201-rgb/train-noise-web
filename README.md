# 🔔 Train Noise Map

An interactive map tool that helps home buyers check train noise impact before purchasing a property.

## What it does

Enter any US address and the app will show you:

- **🚂 Nearby rail crossings** — active train tracks and grade crossings within your area
- **📢 Horn zones** — federally-required horn-blowing zones where trains must sound their horn
- **🌙 Nighttime road noise** — estimated noise levels from nearby roads during night hours
- **🛰️ Satellite view** — toggle between dark map and satellite imagery

Built for use alongside [Redfin](https://www.redfin.com) listings, with a companion [Chrome extension](https://github.com/wk19910201-rgb/train-noise-extension) that adds a one-click **Train Noise** button directly on Redfin property pages.

## Data Sources

- Rail and crossing data: [OpenStreetMap](https://www.openstreetmap.org) via Overpass API
- Map tiles: [CARTO](https://carto.com) / [Esri](https://www.esri.com)
- Geocoding: [Nominatim](https://nominatim.openstreetmap.org)
